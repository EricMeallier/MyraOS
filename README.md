## MyraOS
An operating system written in C/ASM!

# Features
- Bootloader
- Paging
- Heap and dynamic memory
- Drivers: Keyboard, Mouse, VGA, RTC, PIT, PATA
- File system with read support (EXT2), I'm planning to add write support in the future
- Processes and multiprocessing
- ELF parsing
- Basic VESA graphics in full HD
